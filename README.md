1. Description

A book keeping app

2. How to run

Install the required packages using

pip install -r requirements.txt

then edit .env file, replace myGmail@gmail.com with your gmail address and myAppPasswordNow with your 16-digit app password (not account password)

then replace Abcd@1234 with the password you want to keep for the account with the username "admin" having admin privileges; your gmail address will be used for verification

then run

python3 main.py

inside the project's root folder and open the http://localhost:5000/ url.
